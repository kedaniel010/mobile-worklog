# 手机网页版工作记录助手

这是一个适合个人使用的零成本手机版记录工具。

## 能做什么

- 新增工作记录
- 编辑任务
- 标记完成 / 改回未完成
- 搜索任务
- 按未完成 / 已完成 / 高优先级 / 已设提醒筛选
- 导出 CSV
- 数据保存在手机浏览器本地

## 适合你的原因

- 不需要微信云开发
- 不需要服务器
- 不需要付费
- 手机浏览器打开就能用

## 怎么在手机上用

### 方式一：先在电脑浏览器打开看效果

直接打开 `index.html`。

### 方式二：放到手机浏览器里长期使用

最稳的做法是把这套静态文件放到一个免费静态托管平台，然后在手机浏览器打开。

常见免费静态托管：

- GitHub Pages
- Netlify
- Cloudflare Pages

部署后，你在手机上：

1. 打开网址
2. 用浏览器菜单选择 `添加到主屏幕`
3. 以后就可以像 App 一样点桌面图标进入

## 重要说明

- 数据默认只保存在当前浏览器本地
- 换手机、换浏览器、清缓存后，记录不会自动同步
- 这是零成本方案，所以不包含真正的微信消息推送提醒
