# 手机网页版工作记录助手

这是一个适合个人使用的零成本手机版记录工具。

## 能做什么

- 新增工作记录
- 编辑任务
- 标记完成 / 改回未完成
- 搜索任务
- 按未完成 / 已完成 / 高优先级 / 已设提醒筛选
- 导出 CSV
- 导出 JSON 备份
- 导入 JSON 恢复
- 数据保存在手机浏览器本地

## 适合你的原因

- 不需要微信云开发
- 不需要服务器
- 不需要付费
- 手机浏览器打开就能用

## 手机上怎么用

你已经免费发布到了 GitHub Pages，可以直接用这个网址：

[https://kedaniel010.github.io/mobile-worklog/](https://kedaniel010.github.io/mobile-worklog/)

在手机上：

1. 用浏览器打开上面这个网址
2. 通过浏览器菜单选择 `添加到主屏幕`
3. 以后就能像 App 一样从桌面点开

## 怎么备份和恢复

### 备份

1. 打开网页
2. 在 `备份与恢复` 区域点 `导出备份`
3. 会下载一个 `.json` 文件

### 恢复

1. 打开网页
2. 在 `备份与恢复` 区域点 `导入恢复`
3. 选择之前导出的 `.json` 文件
4. 系统会自动恢复记录

## 重要说明

- 数据默认只保存在当前浏览器本地
- 换手机、换浏览器、清缓存后，不会自动同步
- 所以建议你定期导出一个备份 JSON 文件
